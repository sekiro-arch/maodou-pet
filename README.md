# 帽兜

`帽兜` 是一个适用于 Codex 的自定义桌宠项目。

这个仓库整理了桌宠成品、参考素材和动作预览，方便备份、分享和后续继续调整。

## 目录结构

- `package/maodou/`
  - `pet.json`: Codex 桌宠清单文件
  - `spritesheet.webp`: 桌宠动作图集
- `qa/`
  - `contact-sheet.png`: 全动作总览图
  - `previews/`: 各动作 GIF 预览
- `source/`
  - 初始参考图片与 GIF 素材

## 使用方法

将 `package/maodou/` 整个文件夹复制到 Codex 的桌宠目录下即可使用。

Windows 默认目录通常为：

```text
C:\Users\<你的用户名>\.codex\pets\maodou
```

## 说明

- 桌宠名称：`帽兜`
- 当前版本保留了轻微跳跃动作，而不是大幅起跳
- 项目以个人使用和备份为主
